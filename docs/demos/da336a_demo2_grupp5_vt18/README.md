# README

### Virtual environment
``` 
$ python3 -m venv venv
$ source venv/bin/activate
```
### Installing requirements
```
(venv) $ pip install -r requirements.txt
```

### Running the app
```
$ export FLASK_APP=hello.py
$ flask run
```

### OPTIONAL: Running the app debug mode
```
$ export FLASK_APP=app.py
$ export FLASK_DEBUG=True
$ flask run
```
